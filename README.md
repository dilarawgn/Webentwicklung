# Webentwicklung 

Webentwicklung Projekt: PickMySheet

Website mit einem Zufallsgenerator Tool:
Auf meiner Seite kann man verschiedene Wörter eintragen, die als Los in einem Topf in der Mitte „reingelegt“ werden. Dabei ist es völlig egal um was für ein Thema es sich handelt. Nachdem man alle Wörter eingetragen hat, drückt man Start um das Programm einen dieser Lose aussuchen zu lassen. Jetzt wird einem das im Zufallsgenerator gewählte Wort angezeigt und nachdem dieses Wort geschlossen wird, wird es aus dem Topf ausgetragen. Die anderen angegebenen Wörter werden jedoch weiter behalten.
Das Programm ist dafür da um bei bestimmten Entscheidungen das Los ein Wort aussuchen zu lassen oder bestimmte Reihenfolgen anzugeben.
Die ausgewählten Wörter werden an der Seite in einer Liste gespeichert, um die angegbene Reihenfolge bei Schließen der Seite wieder anschauen und umbearbeiten zu können.
Dabei wird immer die zuletzt bearbeitete Liste gespeichert.
